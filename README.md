## Project Overview
This project analyzes HR data to identify employee attrition trends and key workforce insights using Power BI.

## Dataset
- Employee records including age, department, salary, experience, and attrition
- Cleaned and transformed using Power Query

## Key Insights
- Overall attrition rate and employee count
- Department-wise and job role-wise attrition
- Impact of salary, age, and experience on attrition
- Gender-based workforce distribution

## Tools & Technologies
- Power BI
- Power Query
- DAX
